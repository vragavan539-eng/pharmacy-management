# 💊 PharmAI - Pharmacy Management System

Full-stack AI-powered pharmacy management system.

## Setup

### Backend
```bash
cd backend
npm install
cp .env.example .env   # Add ANTHROPIC_API_KEY
node seed.js           # Demo data
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm start
```
